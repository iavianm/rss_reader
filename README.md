# RSS News Reader

**The application allows you to subscribe and read news from RSS news channels**

* Ruby version 3.0.2
* Rails version 6.1.7
* PostgreSQL
* Docker
* * *
* Start project

1. docker-compose build
2. docker-compose up

* * *

**[Link to the demo project](https://iavianm-rss-reader.herokuapp.com/)**
